## About The Project

Photo Share

Social media website made using React, Redux, Redux-Thunk, CSS and Material UI for the frontend. Node, Express and MongoDB for the backend. This repository is stricly for the front end

Features:
* Authetication
* Post photos with descriptions on your account
* Like others posts
* View other profiles
* Post recommendation feature based on the post you are viewing

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Muqto/Photo-Share-client.git
   ```
2. cd into the new folder and install NPM packages
   ```sh
   npm install
   ```
3. To run the React project
    ```sh
   npm start
   ```
